D.R.E.A.M. — Memory Slice

Memory Playbook (9–5)
  • Capture fast: jot the item in your system immediately.
  • Prime review (10–20 min): a short pass to stabilize details.
  • Same-day follow-up: one quick revisit later today.
  • Early spacing: 1d → 3d → 7d → then weekly/bi-weekly.
  • Micro-reviews: 30–90s is enough; don’t re-learn from scratch.
  • Workday guardrails: batch small cards, cap to ~3–5 sessions/day.
Tip: turn Normalize x by λ on — cliffs align at x/λ≈1 so you can compare items fairly.
—
Memory Math (when to review)
\( R(t)=\exp\!\big[-(\lambda t)^{\beta}\big] \)
Pick a target recall \(R^\*\) and schedule when R falls to it:
\( t^\*=\dfrac{[-\ln R^\*]^{1/\beta}}{\lambda} \)
With Normalize on: \( x/\lambda=t\lambda=[-\ln R^\*]^{1/\beta} \). For β≈1: R*=0.60 → x/λ≈0.51; R*=1/e → x/λ=1; R*=0.20 → x/λ≈1.61.
How to read this plot
  • Curve: each slice shows retention for one item/age.
  • β (shape): low β = steeper early drop; high β = gentler start.
  • λ (cliff): larger λ → earlier cliff (review sooner).
  • Cliff line: R=1/e ≈ 0.37 marks a clear loss of detail.
Use the slider to see how age (or item difficulty) shifts β and λ.
