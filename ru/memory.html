D.R.E.A.M. — Срез памяти

Память: практические советы
  • Фиксируйте сразу: запишите мысль/деталь без промедления.
  • Пробный повтор через 10–20 мин: короткий взгляд, чтобы «сцепить» детали.
  • Ещё один повтор сегодня: быстрый проход во второй половине дня.
  • Дальше — растяжение: 1д → 3д → 7д, затем раз в неделю/две.
  • Микроповторы 30–90 с: достаточно освежить, не переучивайте с нуля.
  • Гигиена рабочего дня: группируйте мелкие задачи; 3–5 коротких сессий в день.
Совет: включите Нормировать x по λ — «обрывы» совпадают при x/λ≈1 и элементы проще сравнивать.
—
Математика памяти (когда повторять)
\( R(t)=\exp\!\big[-(\lambda t)^{\beta}\big] \)
Выберите целевой уровень \(R^\*\) и запланируйте момент, когда R опустится до него:
\( t^\*=\dfrac{[-\ln R^\*]^{1/\beta}}{\lambda} \)
При включённой Нормировке: \( x/\lambda=t\lambda=[-\ln R^\*]^{1/\beta} \). Для β≈1: R*=0.60 → x/λ≈0.51; R*=1/e → x/λ=1; R*=0.20 → x/λ≈1.61.
Как читать этот график
  • Кривая: каждый срез — удержание для одного элемента/возраста.
  • β (форма): низкая β = резкий стартовый спад; высокая β = мягкий старт.
  • λ (обрыв): больше λ → обрыв раньше (повтор раньше).
  • Линия обрыва: R=1/e ≈ 0.37 — заметная потеря деталей.
Используйте ползунок, чтобы увидеть, как возраст (или сложность) сдвигает β и λ.
